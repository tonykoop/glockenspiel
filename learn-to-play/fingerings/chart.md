# Glockenspiel Bar Layout Reference

Standard C-major layout (chromatic bars in upper row, diatonic in lower row):

```
   в”Ңв”Җв”Җв”Җв”җ в”Ңв”Җв”Җв”Җв”җ   в”Ңв”Җв”Җв”Җв”җ в”Ңв”Җв”Җв”Җв”җ в”Ңв”Җв”Җв”Җв”җ
   в”ӮC# в”Ӯ в”ӮD# в”Ӯ   в”ӮF# в”Ӯ в”ӮG# в”Ӯ в”ӮA# в”Ӯ   вҶҗ Upper row (sharps/flats)
   в””в”Җв”Җв”Җв”ҳ в””в”Җв”Җв”Җв”ҳ   в””в”Җв”Җв”Җв”ҳ в””в”Җв”Җв”Җв”ҳ в””в”Җв”Җв”Җв”ҳ
 в”Ңв”Җв”Җв”Җв”җ в”Ңв”Җв”Җв”Җв”җ в”Ңв”Җв”Җв”Җв”җ в”Ңв”Җв”Җв”Җв”җ в”Ңв”Җв”Җв”Җв”җ в”Ңв”Җв”Җв”Җв”җ в”Ңв”Җв”Җв”Җв”җ
 в”Ӯ C в”Ӯ в”Ӯ D в”Ӯ в”Ӯ E в”Ӯ в”Ӯ F в”Ӯ в”Ӯ G в”Ӯ в”Ӯ A в”Ӯ в”Ӯ B в”Ӯ   вҶҗ Lower row (naturals)
 в””в”Җв”Җв”Җв”ҳ в””в”Җв”Җв”Җв”ҳ в””в”Җв”Җв”Җв”ҳ в””в”Җв”Җв”Җв”ҳ в””в”Җв”Җв”Җв”ҳ в””в”Җв”Җв”Җв”ҳ в””в”Җв”Җв”Җв”ҳ
   C5   D5   E5   F5   G5   A5   B5

Tunes in this songbook use:  C5  D5  E5  F5  G5  A5  (+ C#5, F#5 for some tunes)
```

## Mallet technique
- Strike the center of each bar for maximum resonance.
- Use a slight wrist flick вҖ” don't press down after impact.
- Alternate mallets: R for odd beats, L for even beats (adjust as needed).
- Dampen with a finger on the bar edge only when a short note is needed.
