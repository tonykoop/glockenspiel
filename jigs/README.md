## Pilot Gate

Build only JD-010 through JD-050 for the C5/A5/C7 pilot. JD-060 follows after pilot bars are trimmed. JD-070 follows after the durometer is selected. JD-070 must be ergonomically verified before any walnut is cut.

See `../jig-decision.md` for full pass/fail criteria and the rejected fixture options.
